ASCII Table Explanation

ASCII (American Standard Code for Information Interchange) is a character encoding standard used for representing text in computers. It includes 128 symbols, including English letters, numbers, and special characters. Each symbol is represented by an integer value ranging from 0 to 127.

ASCII is widely used in programming and data interchange, providing a standardized way to represent characters, making it easier to handle text in various applications.


Code Examples in C

1. Printing ASCII Characters:

#include <stdio.h>

int main() {
    printf("ASCII characters from 32 to 126:\n");
    for (int i = 32; i < 127; i++) {
        printf("ASCII %d: %c\n", i, i);
    }
    return 0;
}

Explanation: This program prints all printable ASCII characters from 32 to 126 using a for loop.

2. Converting a Character to its ASCII Value:

#include <stdio.h>

int main() {
    char character;
    printf("Enter a character: ");
    scanf("%c", &character);
    printf("The ASCII value of '%c' is: %d\n", character, character);
    return 0;
}

Explanation: This program prompts the user to enter a character and prints its ASCII value using %d.

3. Using ASCII for Input Validation:

#include <stdio.h>

int main() {
    char input;
    printf("Enter a character (A-Z or a-z): ");
    scanf("%c", &input);
    
    // Check if the input character is a letter
    if ((input >= 65 && input <= 90) || (input >= 97 && input <= 122)) {
        printf("'%c' is a valid letter.\n", input);
    } else {
        printf("'%c' is not a valid letter.\n", input);
    }
    return 0;
}

Explanation: This program checks if the input character is an uppercase or lowercase letter using ASCII values.

4. Finding the ASCII Value of Digits:

#include <stdio.h>

int main() {
    char digit;
    printf("Enter a digit (0-9): ");
    scanf("%c", &digit);
    
    if (digit >= '0' && digit <= '9') {
        printf("The ASCII value of '%c' is: %d\n", digit, digit);
    } else {
        printf("'%c' is not a valid digit.\n", digit);
    }
    return 0;
}

Explanation: This program verifies if the input character is a digit and prints its ASCII value.
