Format Specifiers in C

  • %d:
    • Used to print integers.
    • Example: int num = 10; printf("%d", num); // Output: 10
    • Another Example: int a = 5, b = 10; printf("Sum: %d", a + b); // Output: Sum: 15
    • Yet Another Example: int x = -20; printf("Negative Number: %d", x); // Output: Negative Number: -20
    Examples in C: 
    int main() {
    int num = 10;
    printf("%d\n", num); // Output: 10

    int a = 5, b = 10;
    printf("Sum: %d\n", a + b); // Output: Sum: 15

    int x = -20;
    printf("Negative Number: %d\n", x); // Output: Negative Number: -20
}
  • %c:
    • Used to print a single character.
    • Example: char letter = 'A'; printf("%c", letter); // Output: A
    • Another Example: char symbol = '#'; printf("Symbol: %c", symbol); // Output: Symbol: #
    • Yet Another Example: char first = 'C'; char second = 'P'; printf("Letters: %c%c", first, second); // Output: Letters: CP
    Examples in C: 
    int main() {
    char letter = 'A';
    printf("%c\n", letter); // Output: A

    char symbol = '#';
    printf("Symbol: %c\n", symbol); // Output: Symbol: #

    char first = 'C', second = 'P';
    printf("Letters: %c%c\n", first, second); // Output: Letters: CP
}
  • %s:
    • Used to print a string.
    • Example: char str[] = "Hello"; printf("%s", str); // Output: Hello
    • Another Example: char name[] = "Alice"; printf("Name: %s", name); // Output: Name: Alice
    • Yet Another Example: char greeting[] = "Welcome to C programming!"; printf("Greeting: %s", greeting); // Output: Greeting: Welcome to C programming!
    Examples in C: 
    int main() {
    char str[] = "Hello";
    printf("%s\n", str); // Output: Hello

    char name[] = "Alice";
    printf("Name: %s\n", name); // Output: Name: Alice

    char greeting[] = "Welcome to C programming!";
    printf("Greeting: %s\n", greeting); // Output: Greeting: Welcome to C programming!
}
  • %llu:
    • Used to print unsigned long long integers.
    • Example: unsigned long long largeNum = 123456789012345; printf("%llu", largeNum); // Output: 123456789012345
    • Another Example: unsigned long long bigNum = 18446744073709551615ULL; printf("Big Number: %llu", bigNum); // Output: Big Number: 18446744073709551615
    • Yet Another Example: unsigned long long count = 1000000000; printf("Count: %llu", count); // Output: Count: 1000000000
    Examples in C: 
    int main() {
    unsigned long long largeNum = 123456789012345;
    printf("%llu\n", largeNum); // Output: 123456789012345

    unsigned long long bigNum = 18446744073709551615ULL;
    printf("Big Number: %llu\n", bigNum); // Output: Big Number: 18446744073709551615

    unsigned long long count = 1000000000;
    printf("Count: %llu\n", count); // Output: Count: 1000000000
}
