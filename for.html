Importance and Uses of For Loop in C++

In **C++**, the for loop is a control flow statement used to repeat a block of code a certain number of times. It is especially useful when the number of iterations is known in advance.

Importance of for Loop in C++:

  • Efficiency: The for loop allows repeating tasks without writing the same code multiple times.
  • Easy to Understand: It is a clear and concise way to loop through arrays, ranges, or sequences of values.
  • Control: It gives precise control over the number of iterations and how the loop progresses.

How to Write a for Loop:

The for loop has three main parts: initialization, condition, and increment/decrement. It starts by initializing a variable, checks the condition before each iteration, and increments or decrements the variable after each iteration.

Syntax:

for (initialization; condition; increment/decrement) {
    // loop body
}

Examples:

Example 1: Print numbers from 1 to 5
#include <iostream>

int main() {
    for (int i = 1; i <= 5; i++) {
        std::cout << i << " ";
    }
    return 0;
}
Example 2: Sum of first 10 numbers
#include <iostream>

int main() {
    int sum = 0;
    for (int i = 1; i <= 10; i++) {
        sum += i;
    }
    std::cout << "Sum: " << sum << std::endl;
    return 0;
}
Example 3: Looping through an array
#include <iostream>

int main() {
    int numbers[] = {10, 20, 30, 40, 50};
    
    for (int i = 0; i < 5; i++) {
        std::cout << "Element at index " << i << ": " << numbers[i] << std::endl;
    }
    
    return 0;
}
Example 4: Using break in a for loop
#include <iostream>

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            break;
        }
        std::cout << i << " ";
    }
    return 0;
}
Example 5: Using continue in a for loop
#include <iostream>

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            continue;
        }
        std::cout << i << " ";
    }
    return 0;
}
