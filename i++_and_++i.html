Increment and Decrement Operators

  • i++ (Post-increment):
    • Increases the value of i by 1 after its current value is used.
    • Example: x = i++ assigns x the current value of i, then increments i by 1.
    Example in C++:

        int i = 5;
    int x = i++;
    
    // x is assigned 5, then i becomes 6.
  • ++i (Pre-increment):
    • Increases the value of i by 1 before its current value is used.
    • Example: x = ++i increments i first, then assigns the new value to x.
    Example in C++:

        int i = 5;
    int x = ++i;
    
    // i becomes 6, then x is assigned 6.
  • i-- (Post-decrement):
    • Decreases the value of i by 1 after its current value is used.
    • Example: x = i-- assigns x the current value of i, then decrements i by 1.
    Example in C++:

        int i = 5;
    int x = i--;
    
    // x is assigned 5, then i becomes 4.
  • --i (Pre-decrement):
    • Decreases the value of i by 1 before its current value is used.
    • Example: x = --i decrements i first, then assigns the new value to x.
    Example in C++:

        int i = 5;
    int x = --i;
    
    // i becomes 4, then x is assigned 4.
