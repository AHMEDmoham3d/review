Power in Programming

The term Power refers to the operation of raising a number (the base) to the exponent (the power). It is an essential mathematical operation in programming, used in various applications such as scientific computations, graphics programming, and data analysis.

The power operation is often represented as base^exponent, which means multiplying the base by itself for the number of times indicated by the exponent. For example, 2^3 (2 raised to the power of 3) equals 8.

Programming Examples

1. Using the `pow()` Function in C:

#include <stdio.h>
#include <math.h>

int main() {
    double base = 2.0;
    double exponent = 3.0;

    double result = pow(base, exponent);  // 2^3 = 8
    printf("%.2f raised to the power of %.2f is %.2f\n", base, exponent, result);
    
    return 0;
}

Explanation: This program uses the pow() function from the math.h library to calculate 2 raised to the power of 3.

2. Power Operation in Python:

base = 3
exponent = 4

result = base ** exponent  # 3^4 = 81
print(f"{base} raised to the power of {exponent} is {result}")

Explanation: In Python, the ** operator is used to raise 3 to the power of 4, resulting in 81.

3. Calculating Power using Loops in JavaScript:

function power(base, exponent) {
    let result = 1;
    for (let i = 0; i < exponent; i++) {
        result *= base; // Multiply base for exponent times
    }
    return result;
}

console.log(power(5, 3)); // 5^3 = 125

Explanation: This JavaScript function calculates the power of a number by multiplying the base by itself for the given exponent.
