Functions in Programming


  • What is a Function?

    A function is a block of code that performs a specific task. It can take inputs, called parameters, and can return a value.

  • Importance of Functions:

    • Code Reusability: Functions allow you to reuse code in multiple places without rewriting it.
    • Organization: They help organize code into manageable sections.
    • Maintainability: Functions make it easier to update code.
  • How to Write a Function:

    In C++, a function is defined using the following syntax:

    return_type function_name(parameters) {
    // body of the function
    return value; // if applicable
}
  • Types of Functions:

    • Void Functions: Do not return a value.
      • Example in C++: 
      void greet() {
    cout << "Hello, World!" << endl;
}
    • Value Returning Functions: Return a value.
      • Example in C++: 
      int add(int a, int b) {
    return a + b;
}
    • Parameterized Functions: Take parameters.
      • Example in C++: 
      void printSum(int a, int b) {
    cout << "Sum: " << a + b << endl;
}
  • How to Call a Function:

    You call a function by using its name followed by parentheses.
    If it takes parameters, pass the values inside the parentheses.


    
greet(); // Calls the greet function
int result = add(5, 10); // Calls the add function and stores the result
printSum(3, 4); // Calls the printSum function
