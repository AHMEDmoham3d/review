For Loop in C Programming

The for loop in C is used to repeat a block of code a certain number of times. It is mainly used when the number of iterations is known beforehand.


Syntax:


for (initialization; condition; increment) {
    // Block of code to execute
}

Explanation of Parts:


  • Initialization: This part initializes the loop counter, usually declared as an integer.
  • Condition: The condition is evaluated before each iteration. If it's true, the loop continues; if false, it stops.
  • Increment/Decrement: After each iteration, the loop counter is updated, usually incremented or decremented.

Examples:


  • Example 1: Print Numbers from 1 to 5 
    #include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    return 0;
}

    Output: 1 2 3 4 5

  • Example 2: Sum of Numbers from 1 to 10 
    #include <stdio.h>

int main() {
    int sum = 0;
    for (int i = 1; i <= 10; i++) {
        sum += i;
    }
    printf("Sum: %d", sum);
    return 0;
}

    Output: Sum: 55

  • Example 3: Print Even Numbers between 1 and 10 
    #include <stdio.h>

int main() {
    for (int i = 2; i <= 10; i += 2) {
        printf("%d ", i);
    }
    return 0;
}

    Output: 2 4 6 8 10
