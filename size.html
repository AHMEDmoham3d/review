Size of Vector in C++

  • What is Size?
    • The size of a vector in C++ refers to the number of elements currently stored in the vector.
    • You can obtain the size of a vector using the size() member function.
  • Example of Using Size:
    Example in C++:

    #include <iostream>
#include <vector>

int main() {
    std::vector myVector;

    myVector.push_back(10); // Add element
    myVector.push_back(20); // Add element
    myVector.push_back(30); // Add element

    std::cout << "Size of vector: " << myVector.size() << std::endl; // Output the size

    return 0;
}
  • Dynamic Changes:
    • The size of a vector can change dynamically as elements are added or removed.
    • Each time you call push_back(), the size increases by one.
    • When you call pop_back(), the size decreases by one.
  • Example of Dynamic Changes:
    Example in C++:

    #include <iostream>
#include <vector>

int main() {
    std::vector myVector;

    myVector.push_back(10);
    myVector.push_back(20);
    myVector.push_back(30);
    std::cout << "Size after adding elements: " << myVector.size() << std::endl; // Outputs 3

    myVector.pop_back(); // Remove the last element
    std::cout << "Size after removing an element: " << myVector.size() << std::endl; // Outputs 2

    return 0;
}
