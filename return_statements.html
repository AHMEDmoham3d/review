Return Statements in C++

  • return 0:
    • Indicates successful execution of the program.
    • Used in the main function to signify that the program ran without errors.
    Example in C++: 
    int main() {
    // Code logic here
    return 0; // Indicating successful execution
}
  • return 1:
    • Can be used to indicate a general error or abnormal termination.
    • It's common to use this value to signify that the program encountered a problem.
    Example in C++: 
    int main() {
    // Simulating an error
    if (/* error condition */) {
        return 1; // Indicating an error
    }
    return 0; // Success
}
  • return -1:
    • Similar to return 1, but often used to indicate a specific type of error.
    • Negative return values can help differentiate between error types.
    Example in C++: 
    int main() {
    // Specific error condition
    if (/* specific error */) {
        return -1; // Indicating a specific error
    }
    return 0; // Success
}
  • return true:
    • Represents a true condition in functions that return a boolean value.
    • Commonly used in conditional checks.
    Example in C++: 
    bool isEven(int number) {
    return number % 2 == 0; // Returns true if number is even
}

int main() {
    if (isEven(4)) {
        // Do something
    }
    return 0;
}
  • return false:
    • Represents a false condition in boolean-returning functions.
    • Indicates that a certain condition has not been met.
    Example in C++: 
    bool isPositive(int number) {
    return number > 0; // Returns false if number is not positive
}

int main() {
    if (!isPositive(-5)) {
        // Handle the negative case
    }
    return 0;
}
