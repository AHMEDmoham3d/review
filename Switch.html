Switch Statement in C++

  • What is a Switch Statement?
    • The switch statement is a control structure used to handle multiple conditions based on the value of a single variable.
    • It is an alternative to multiple if-else statements when we need to compare a variable to a set of constant values.
    • Switch statements make the code cleaner and more readable, especially when dealing with numerous options.
  • Basic Syntax of Switch:
    Example in C++:

        switch (expression) {
        case value1:
            // code block for value1
            break;
        case value2:
            // code block for value2
            break;
        default:
            // code block if no cases match
    }
  • Example 1 - Day of the Week:
    • This example shows how we can print the name of a day based on a number input.
    Example in C++:

        int day = 3;

    switch(day) {
        case 1:
            cout << "Monday";
            break;
        case 2:
            cout << "Tuesday";
            break;
        case 3:
            cout << "Wednesday";
            break;
        case 4:
            cout << "Thursday";
            break;
        case 5:
            cout << "Friday";
            break;
        case 6:
            cout << "Saturday";
            break;
        case 7:
            cout << "Sunday";
            break;
        default:
            cout << "Invalid day";
    }
  • Example 2 - Simple Calculator:
    • This is a simple calculator using the switch statement to select the operation based on user input.
    Example in C++:

        char operation;
    float num1, num2;

    cout << "Enter operator (+, -, *, /): ";
    cin >> operation;
    cout << "Enter two numbers: ";
    cin >> num1 >> num2;

    switch(operation) {
        case '+':
            cout << num1 + num2;
            break;
        case '-':
            cout << num1 - num2;
            break;
        case '*':
            cout << num1 * num2;
            break;
        case '/':
            if(num2 != 0)
                cout << num1 / num2;
            else
                cout << "Error: Division by zero!";
            break;
        default:
            cout << "Invalid operator";
    }
  • Example 3 - Grade Evaluation:
    • This example evaluates a student's grade and provides feedback based on the letter grade.
    Example in C++:

        char grade = 'B';

    switch (grade) {
        case 'A':
            cout << "Excellent!";
            break;
        case 'B':
            cout << "Good!";
            break;
        case 'C':
            cout << "Fair!";
            break;
        case 'D':
            cout << "Poor!";
            break;
        case 'F':
            cout << "Failed!";
            break;
        default:
            cout << "Invalid grade!";
    }
  • Key Features of Switch:
    • The break statement is used to exit the switch block once a case has been executed.
    • The default case is optional but useful for handling cases where none of the defined cases match the value.
    • It works with integers, characters, and enumeration values, but not floating-point values.
  • Switch vs. If-Else:
    • Switch: More efficient and cleaner when working with multiple fixed values. Easier to read.
    • If-Else: More flexible and can handle complex conditions, not just constant values.
