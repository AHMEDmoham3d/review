Logical Operators

  • && (Logical AND):
    • Means "and" in logic.
    • Used to check if both conditions are true. If both are true, it returns true; otherwise, it returns false.
    • Example: true && false returns false because one of the sides is false.
    Example in C++:

    int x = 5, y = 10;
if (x > 0 && y > 5) {
    cout << "Both conditions are true!";
}
  • || (Logical OR):
    • Means "or" in logic.
    • Returns true if at least one of the conditions is true, and returns false only if all conditions are false.
    • Example: true || false returns true because one side is true.

    Example in C++:

    int x = -5, y = 10;
if (x > 0 || y > 5) {
    cout << "At least one condition is true!";
}
  • ! (Logical NOT):
    • Used to negate a condition. If the condition is true, it returns false, and vice versa.
    • Example: !true returns false.
    Example in C++:

    bool is_raining = false;
if (!is_raining) {
    cout << "It's not raining!";
}
  • ^ (Logical XOR - Exclusive OR):
    • Used to perform a comparison where it returns true if one condition is true and the other is false.
    • Returns false if both conditions are true or both are false.
    • Example: true ^ false returns true, but true ^ true returns false.
    Example in C++:

    bool a = true, b = false;
if (a ^ b) {
    cout << "XOR condition is true!";
}
